About Python Level 1¶

The League’s Python Level 1 curriculum introduces students to the core feature of Python using professional tools like Github and Visual Studio Code. The course is aligned with the PCEP™ – Certified Entry-Level Python Programmer exam from the Python Institue, so students who have completed this course will be familiar with all of material on the test.

(2)¶\[\begin{split} (a + b)^2 &= (a + b)(a + b) \\ &= a^2 + 2ab + b^2 \end{split}\]
